This repo is made while implementing different kinds of RAGs, strating from simple (Naive) RAG.  

These RAGs will be implemented,

1. Naive RAG (single PDF as a source) + Evals
2. Naive RAG (Python and C code as a source, multiple files)
3. Naive RAG with tabular data, OCR, multiple files
4. Experiment with [document retrievers](https://docs.langchain.com/oss/python/integrations/retrievers)
5. Explore Chonkie for semantic based chunking
6. RAG for whole repo or knowledge base level
7. GraphRAG / Hierarchial RAG
8. Hybrid RAG

Alongside, we'll learn 

- Different chunking strategies
- RAG evaluation
- Reranking
- Query rewriting

many more
